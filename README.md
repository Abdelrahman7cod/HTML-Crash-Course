# 📘 HTML Crash Course

Welcome to the **HTML Crash Course**.  
This guide introduces the basics of **HTML (HyperText Markup Language)** and helps beginners understand how web pages are built.

---

# 📌 What is HTML?

**HTML** stands for **HyperText Markup Language**.

It is the language used to **create the structure of web pages**.

HTML is **not a programming language**.  
It is a **Markup Language** that uses **tags** to describe content.

Examples of what HTML can define:

- Headings
- Paragraphs
- Images
- Links
- Lists
- Forms

---

# 🧱 Basic HTML Page Structure

Every HTML page follows a basic structure:

```html
<!DOCTYPE html>
<html>
<head>
<title>My Page</title>
</head>

<body>
<h1>Hello World</h1>
</body>

</html>
```

### Explanation

| Tag | Description |
|----|----|
| `<!DOCTYPE html>` | Defines the document as HTML5 |
| `<html>` | Root element of the page |
| `<head>` | Contains metadata |
| `<body>` | Contains visible page content |

---

# 🏷 Understanding HTML Tags

Most HTML elements follow this pattern:

```html
<p>This is a paragraph</p>
```

Structure:

- Opening Tag
- Content
- Closing Tag

Example:

```html
<h1>Hello</h1>
```

Some elements **do not require a closing tag**, like:

```html
<img src="image.jpg">
```

---

# 📝 Essential HTML Tags

## 1️⃣ Headings

```html
<h1>Main Title</h1>
<h2>Sub Title</h2>
<h3>Section</h3>
```

HTML provides **6 heading levels** from **h1 to h6**.

---

## 2️⃣ Paragraph

```html
<p>This is a paragraph.</p>
```

---

## 3️⃣ Links

```html
<a href="https://google.com">Go to Google</a>
```

`href` is an **attribute** that specifies the destination link.

---

## 4️⃣ Images

```html
<img src="image.jpg" alt="My Image">
```

| Attribute | Description |
|------|------|
| `src` | Image source |
| `alt` | Alternative text |

---

## 5️⃣ Lists

### Unordered List

```html
<ul>
<li>HTML</li>
<li>CSS</li>
<li>JavaScript</li>
</ul>
```

### Ordered List

```html
<ol>
<li>Step 1</li>
<li>Step 2</li>
</ol>
```

---

# 🧩 HTML Attributes

Attributes provide **extra information about elements**.

Example:

```html
<input type="text" placeholder="Your Name">
```

Common attributes include:

- `type`
- `href`
- `src`
- `alt`
- `placeholder`

---

# 📋 HTML Forms

Forms are used to **collect user input**.

```html
<form>

<input type="text" placeholder="Name">

<input type="email" placeholder="Email">

<button type="submit">Send</button>

</form>
```

Common input types:

- text
- email
- password
- number
- checkbox
- radio
- file
- date

---

# 📦 div vs span

```html
<div>Block Element</div>
<span>Inline Element</span>
```

| Element | Type |
|------|------|
| div | Block Element |
| span | Inline Element |

---

# 💬 HTML Comments

Comments are used to leave notes in code.

```html
<!-- This is a comment -->
```

They **do not appear on the webpage**.

---

# 🧠 Useful HTML Tricks

### Open links in a new tab

```html
<a href="https://google.com" target="_blank">
```

---

### Mobile responsive viewport

```html
<meta name="viewport" content="width=device-width, initial-scale=1.0">
```

---

### Lazy loading images

```html
<img src="image.jpg" loading="lazy">
```

---

### Semantic HTML

```html
<header></header>
<main></main>
<footer></footer>
```

Semantic tags improve **SEO** and **code readability**.

# 🧪 Mini Project

Create a simple page containing:

- Heading
- Paragraph
- Image
- List
- Link
- Form

Example:

```html
<!DOCTYPE html>
<html>

<head>
<title>My First Page</title>
</head>

<body>

<h1>Welcome</h1>

<p>This is my first HTML page.</p>

<img src="image.jpg" alt="My Image">

<ul>
<li>HTML</li>
<li>CSS</li>
</ul>

<a href="https://google.com">Visit Google</a>

<form>

<input type="text" placeholder="Your Name">

<input type="email" placeholder="Your Email">

<button>Send</button>

</form>

</body>
</html>
```
